# Flask CRM System

## Projektbeschreibung

Dieses Projekt ist ein einfaches CRM-System (Customer Relationship Management), das mit **Flask** entwickelt wurde.

Die Anwendung ermöglicht es, Kunden, Leads, Kontakte, Aufgaben und Termine zu verwalten.  
Zusätzlich stellt das System eine **REST-API** bereit, über die externe Clients (API Clients) auf die Daten zugreifen können.

Das Projekt wurde im Rahmen eines Scrum-Projekts in mehreren Sprints entwickelt.

---

# Technologien

- Python
- Flask
- Jinja2 Templates
- HTML / CSS / Bootstrap
- REST API (JSON)
- Git / GitHub
- Jira (Projektmanagement)

---

# Rollen im System

### Sales User
Kann folgende Funktionen nutzen:

- Kunden verwalten
- Leads verwalten
- Kontakte hinzufügen
- Aufgaben erstellen und bearbeiten
- Termine erstellen
- Dashboard anzeigen

### Admin
Kann zusätzlich:

- Benutzer verwalten
- alle Daten verwalten

### API Client
Externe Systeme können über die REST API auf folgende Ressourcen zugreifen:

- Customers
- Leads
- Contacts
- Tasks
- Appointments
